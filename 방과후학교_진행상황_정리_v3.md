## 📋 현재 정확한 진행 상태 (앱 탭별 진행도)

### 1️⃣ 탭 1: 기초 설정 및 학사일정 (✅ 완료)
- 기준 연도, 학교명, 월 선택, 기본 강사료 단가 설정 완료.
- 공휴일/방과후 미운영일 지정 및 월별 요일별 시수(수업일수) 자동 계산 기능 완료.

### 2️⃣ 탭 2: 기초자료 (학생/프로그램) (✅ 완료)
- 기존 엑셀(수강신청, 학생자격, 개설프로그램) 데이터 업로드 기능 구현.
- 자격 상세 정보에 기반하여 1순위/2순위 지원금 자격 자동 분배.
- **[신규]** 수강신청 데이터를 바탕으로 프로그램별 '신청 학년' 자동 추출 및 표시.
- **[신규]** 프로그램 목록에서 운영요일, 신청학년 등 필터링 검색 기능 추가.
- 금액 표시는 모두 천단위(원) 포맷 적용 완료.

### 3️⃣ 탭 3: 수강료 정산 (✅ 완료)
- 프로그램별 단가 및 수강료/재료비 편집 기능.
- 학생별 가구자격(초3이용권, 기초생계급여수급자, 다자녀 등)에 따른 수강료와 지원금 자동 분리 계산.
- **[신규]** '환급액'과 '환급사유'를 사용자가 직접 엑셀처럼 더블클릭하여 수정할 수 있는 에디터 기능 구현.
- 수정 사항 실시간 반영 및 데이터 로컬 저장(JSON) 완료.

### 4️⃣ 탭 4: 강사료 정산 (✅ 완료)
- 프로그램별/강사별 실제 수업 차시 반영 기능.
- 교육비지원 총액, 수익자 부담액을 정산하여 총 수강료 수입 산출.
- 교육청 강사료 보전금(강사료 지출액이 수입보다 클 경우 부족분 지원) 자동 계산 적용.

### 5️⃣ 탭 5: 통계 (✅ 완료)
- 프로그램별 수강료 수입/지출/보전금 통계 보고서 완료.
- 강사 구분별(내/외부) 요약 통계 완료.
- **[신규]** 3월~내년 2월까지의 월별 누계 및 **재원별 수입/지출 종합 통계(학생 수익자 부담, 교육비지원 상세 항목별, 보전금 등)** 와이드 표 포맷 완성.
- 모든 표의 금액 천단위 콤마 포맷팅 ⚠️오류 해결 및 안정화 완료.
- 모든 페이지에 **엑셀 다운로드 버튼** 기능 적용 완료.

### 1. 월별 데이터 관리 및 검색 기능 추가
- 매월 계산되는 항목(신청, 강사료, 수강료)들을 월별로 저장하고 추적하는 기능 구현.
- 사용자가 원하는 특정 월을 선택하여 과거에 올렸던 자료와 정산 내역을 불러오고 검색할 수 있는 **월별 대시보드 조회 기능** 추가.

### 2. 자료 업로드 형태 및 워크플로우 개선
- 매달 새로운 데이터를 업로드할 때, 이전 달의 데이터(예: 이월된 환급액, 연속 수강생 등)가 어떻게 연동되고 저장될지에 대한 구조 설계 및 UI 개선.
- 월이 바뀌어도 안전하게 기존 데이터를 아카이빙(보존)하고 새 데이터를 처리할 수 있는 형태 마련.

### 3. 세부 기능 고도화
- 사용자 피드백을 반영한 추가적인 통계 항목이나 검색 조건 고도화.
- 전체 데이터의 월간/연간 통합 통계 대시보드 구성 검토.
